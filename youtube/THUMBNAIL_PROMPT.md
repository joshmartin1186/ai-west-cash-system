# YouTube Thumbnail Creation Prompt

> **Use this prompt when working with designers or AI image generators to create thumbnails**

---

## Comprehensive Thumbnail Brief Template

```
THUMBNAIL REQUEST FOR: [Video Title]

VIDEO TOPIC: [Brief description of video content]

PSYCHOLOGY STRATEGY:
- Curiosity Gap Type: [Before/After | Challenge | Contradiction | Novelty | Result]
- Primary Emotion: [Shock | Curiosity | Relief | Excitement | FOMO]
- Core Desire: [Make money | Save time | Achieve status | Remove pain | Get results]

ATTENTION TRIGGERS (Choose 2-3):
1. [Face with emotion: specify exact emotion]
2. [Big numbers: specify which numbers]
3. [Familiar visuals: specify logos/icons]
4. [Other triggers if applicable]

LAYOUT:
[Split Screen | Offset/Asymmetrical | Centered]

COMPOSITION:
- Subject Placement: [Left third | Right third | Center]
- Background: [Describe background style/color]
- Key Visual Elements: [List 2-4 main visual elements]

TEXT (if needed):
- Primary Text: "[2-4 words max]"
- Text Placement: [Top | Bottom | Center | Integrated]
- Text Color: [Specify - must contrast with background]
- Text Style: [Bold | Clean | Impactful]

STYLE REFERENCES:
- Color Palette: [Describe overall color scheme]
- Visual Style: [Clean/Modern | Bold/Dramatic | Professional | Casual]
- Reference Thumbnails: [Link to examples if available]

TECHNICAL SPECS:
- Dimensions: 1280 x 720 pixels
- Format: JPG or PNG
- File Size: Under 2MB
- Safe Zones: No important elements in bottom-right corner (timestamp)

CRITICAL REQUIREMENTS:
✓ Must work on dark mode background
✓ Must be readable at thumbnail size (small mobile screens)
✓ Text must be legible when thumbnail is tiny
✓ Face expression (if used) must be STRONG and match video tone
✓ Maximum 3 attention triggers (don't overload)

DELIVERABLES:
- 3 variations for A/B testing
- High-resolution source file
- Optimized version for YouTube upload
```

## Example Thumbnail Brief: Build Video

```
THUMBNAIL REQUEST FOR: "I Built an AI CRM That Runs Itself"

VIDEO TOPIC: Tutorial showing how to build an intelligent CRM that automatically updates from call transcripts, schedules follow-ups, and generates emails.

PSYCHOLOGY STRATEGY:
- Curiosity Gap Type: Result (showing what was built)
- Primary Emotion: Excitement ("I could do this")
- Core Desire: Save time + Remove pain (manual CRM updates)

ATTENTION TRIGGERS:
1. Big numbers: "47 hours" → "10 minutes"
2. Familiar visuals: Airtable logo, Claude logo, n8n logo
3. Face with emotion: Confident/satisfied expression

LAYOUT: Offset/Asymmetrical

COMPOSITION:
- Subject Placement: Right third of frame
- Background: Gradient (dark blue to purple) or solid dark with subtle pattern
- Key Visual Elements:
  * Me (right side) with confident expression
  * Before/After comparison (left side):
    - Before: Messy spreadsheet screenshot
    - After: Clean dashboard screenshot
  * Tool logos (Airtable, Claude, n8n) integrated naturally

TEXT:
- Primary Text: "Zero Code"
- Text Placement: Top left, above before/after comparison
- Text Color: Bright white or neon green (high contrast)
- Text Style: Bold, modern sans-serif

STYLE REFERENCES:
- Color Palette: Dark background with bright accent colors
- Visual Style: Clean/Modern with tech vibe
- Similar to tech YouTuber style (Fireship, Theo, Josh Tried Coding)

TECHNICAL SPECS:
- Dimensions: 1280 x 720 pixels
- Format: PNG for transparency flexibility
- File Size: Under 2MB
- Safe Zones: Keep tool logos away from bottom-right

CRITICAL REQUIREMENTS:
✓ Must pop on dark YouTube background
✓ "Zero Code" must be instantly readable even at small size
✓ Face expression is confident but not arrogant
✓ Before/After comparison is immediately clear
✓ Tool logos recognizable but not cluttered

DELIVERABLES:
- Variation 1: Face + Before/After + "Zero Code"
- Variation 2: No face, bigger Before/After, "Zero Code" more prominent
- Variation 3: Face center, tools floating around, "Zero Code" bottom
```

## Quick Thumbnail Psychology Reference

### When to Use Each Curiosity Gap:

**Before & After:** Transformation, improvement, progress content
- Example: "30 subs → 10K subs" or "Manual → Automated"

**Challenge:** "I did X for Y days" or intense/difficult tasks
- Example: "I tried every CRM tool" or "Built 10 systems in 10 days"

**Contradiction:** Breaks expectations, shouldn't be possible
- Example: "$0 to $100K with no ads" or "Automated sales with zero SDRs"

**Novelty:** Something new, weird, or unusual
- Example: "Built CRM in Google Sheets" or "This AI writes my emails"

**Result:** Shows final achievement, makes viewers want to know how
- Example: "15% Reply Rate" or "10 Meetings Per Week"

### When to Use Each Attention Trigger:

**Face:** Any video where personal brand matters, tutorials, commentary
**Famous People:** Only if they actually appear in the video
**Big Numbers:** Money, time, metrics, transformations
**Familiar Visuals:** Tech tools, platforms, recognizable brands
**Aesthetics:** High-production tutorials, professional content
**Movement/Drama:** Action content, challenges, intense topics

## Thumbnail Testing Checklist

Before finalizing, test each thumbnail:

**Visual Tests:**
- [ ] View at actual thumbnail size (small)
- [ ] View on dark mode background
- [ ] View on light mode background
- [ ] Check if text is readable when tiny
- [ ] Verify nothing important in bottom-right corner

**Psychology Tests:**
- [ ] 2-second flash test: Show to someone for 2 seconds, ask what they saw
- [ ] Emotion check: Does face expression match video tone?
- [ ] Curiosity check: Would I click this based on thumbnail alone?
- [ ] Clarity check: Is the main idea immediately obvious?

**Technical Tests:**
- [ ] Correct dimensions (1280 x 720)
- [ ] Under 2MB file size
- [ ] High enough resolution
- [ ] Optimized for upload

**A/B Test Strategy:**
- Create 3 variations
- Test which gets highest CTR
- Note what worked for future thumbnails

## Common Thumbnail Mistakes - Visual Examples

### ❌ TOO CLUTTERED
```
[Face] + [5 logos] + [3 text blocks] + [arrows] + [graphics] = CHAOS
```

### ✅ CLEAN & FOCUSED
```
[Face] + [1-2 key elements] + [Short text] = CLEAR MESSAGE
```

### ❌ UNREADABLE TEXT
```
"Here's How To Build An Automated System In Under An Hour"
(Way too long, won't read when small)
```

### ✅ READABLE TEXT
```
"Zero Code"
(Short, punchy, readable)
```

### ❌ WEAK FACE EXPRESSION
```
Neutral smile (boring, doesn't convey emotion)
```

### ✅ STRONG FACE EXPRESSION
```
Surprised/shocked expression (immediately grabs attention)
```

## Designer Communication Tips

When working with a designer:

1. **Provide visual references:** Link to thumbnails you like
2. **Be specific about emotion:** "Shocked" not "happy"
3. **Specify exact text:** Don't let them paraphrase
4. **Request variations:** Always get 3 versions
5. **Give clear feedback:** "Make text 50% bigger" not "text feels small"
6. **Set deadline:** "Need by [date] for upload"
7. **Include video link:** Let them watch to understand tone

## AI Image Generator Prompts

If using Midjourney, DALL-E, or similar:

```
YouTube thumbnail, professional photography style, [describe subject], [describe setting], bright lighting, shallow depth of field, 16:9 aspect ratio, bold text overlay reading "[your text]", high contrast, modern design, trending on YouTube

Style: Clean and professional
Colors: [specify color palette]
Mood: [confident | excited | dramatic | etc.]
```

**Note:** AI tools struggle with text - always add text in post-processing (Photoshop, Canva, etc.)
